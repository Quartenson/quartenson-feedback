## What to report

Use issues for:

- confusing instructions
- broken tool behavior
- mobile layout problems
- accessibility problems
- result calculation issues
- feature requests
- translation issues

## Before opening an issue

Please include:

- tool or page URL
- device/browser
- what happened
- what you expected
- screenshot if useful

## Important note

Quartenson tools are educational and recreational exercises. They are not medical, diagnostic, or professional assessment tools.
